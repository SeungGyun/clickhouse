# clickhouse
클릭하우스
###도커 실행
docker-compose up -d
###도커 종료
docker-compose down
###도커 종료 초기화
docker-compose down -v
###도커 로그 확인
docker-compose logs -f
###도커 컨테이너 목록 확인
docker ps
###도커 컨테이너 접속
docker exec -it clickhouse bash
